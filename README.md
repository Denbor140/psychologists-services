## 📜 About

**Psychologists Services** a web application for finding psychologists and booking appointments with them online.

It helps users:

- browse a directory of psychologists with detailed information (experience, price, reviews)
- filter psychologists by various criteria
- add psychologists to a favorites list
- register and log in to an account
- submit an appointment request via the booking form

---

## ⚡ Features

- 🧑‍⚕️ Psychologist directory with detailed profiles
- 🔍 Filtering by name (alphabetical), price, and popularity
- ❤️ Favorite psychologists (available to logged-in users only)
- 🔐 Registration and login
- 📋 Consultation booking form with validation
- 🔔 Toast notifications for user actions

---

## 🧰 Tech Stack

- **Framework:** React + Next.js (App Router)
- **Language:** TypeScript
- **Auth:** Firebase Authentication
- **Database:** Realtime Database (by firebase)
- **Data Fetching:** React Query
- **Forms:** React Hook Form + Yup
- **Styling:** CSS Modules
- **Notification:** Toaster
